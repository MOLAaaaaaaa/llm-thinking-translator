LLM Thinking Translator

翻译 ChatGPT、Claude、Gemini 网页端的英文思考内容。默认保持安静，调试信息只在调试模式下出现。

翻译接口

模板变量：{{text}}、{{target}}、{{host}}、{{url}}。返回 JSON 时优先读取 translatedText、translation、text、result。

兼容 OpenAI、OpenRouter、DeepSeek 等 Chat Completions 接口。

自定义选择器，按域名覆盖

示例：{"chatgpt.com":["details","[data-testid*=\\"thinking\\" i]"],"claude.ai":["[class*=\\"thinking\\" i]"]}

黑名单选择器

每行一个 CSS selector。匹配到这些区域会跳过，避免翻译输入框、导航栏、代码块。
